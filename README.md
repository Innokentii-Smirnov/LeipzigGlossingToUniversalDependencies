# Инструкция
Разместите atr_val.py в папке src/translator, а входные файлы - в папке input в корневом каталоге репозитория.
Папки могут располагатьтся в поддиректориях с произвольной глубиной вложения.

Установите stanza:
```
pip install stanza
```

Выполните:
```
python src/main.py
```
Выходные файлы будут созданы в папке output.
Структура поддиректорий будет повторять структуру папки input.

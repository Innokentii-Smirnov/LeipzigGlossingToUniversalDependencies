# Инструкция
Разместите atr_val.py в папке src/translator, а входные файлы - в папке input в корневом каталоге репозитория.
Папки могут располагатьтся в поддиректориях с произвольной глубиной вложения.

Установите stanza:
```
pip install stanza
```

Выполните:
```
python src/main.py
```
Выходные файлы будут созданы в папке output.
Структура поддиректорий будет повторять структуру папки input.

# Преобразование проклитик в префиксы
После порождения файлов JSON к ним можно применить дополнительную программу,
которая заменит проклитики на префиксы, а их лицо и число, а также
возвратность, перенесет на глагол с меткой \[obj\].
Чтобы запустить ее, выполните:
```
python src/add_prefixes.py
```
Собственные лицо и число всякого глагола (независимо от наличия проклитики)
получат метку \[subj\].
В текущей версии это преобразование применяется только к токенам,
но не к составляющим.
